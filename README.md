Project Name : TileCraft – Tiles Gallery Website

**Project Purpose**

TileCraft is a modern tiles gallery web application where users can explore premium tile collections, view details, and discover the perfect aesthetic for their spaces.
This project was built as a Next.js App Router assignment focusing on authentication, protected routes, API integration, and UI libraries.


Live Website : https://tile-craft.vercel.app


**Key Features**
1-Home Page
2-Tiles Gallery
3-Authentication (BetterAuth)
4-Protected Routes
5-My Profile


**NPM Packages Used**

1-@heroui/react– UI components library
2-better-auth– authentication system
3-mongodb– database integration
4-react-hook-form– form handling & validation
5-react-toastify– notifications & alerts
6-react-fast-marquee– smooth marquee animation
7-animate.css– UI animations
8-react-icons– icon library
9-json-server– mock API server
